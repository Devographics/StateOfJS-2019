<span class="first-letter">사</span>실 저희가 이 조사를 하는 건 2018년이 마지막이라고 생각했습니다. 자바스크립트 생태계가 *또 한번* 이 정도로 변하지는 않을테니까요, 그렇지 않나요?

하지만 여러분 모두 아시다시피, 자바스크립트의 변화는 아직 끝나지 않았습니다! 그리하여 올해 설문으로 21,717 분의 응답을 받은 후에 모든 컴포넌트와 차트를 갈아엎고, 코드를 엉망진창으로 짰던 1년 전의 저희를 저주하며, 데이터를 들여다보아야 했습니다. 

몇몇 도움도 얻을 수 있었습니다. 데이터 시각화 전문가로 정말 놀라운 재능을 가진 [Amelia Wattenberger](http://wattenberger.com/) 께서 팀에 객원으로 참여해 저희 데이터에 새 옷을 입혀주셨으며, 그 결과물은 **개요** 섹션에서 확인하실 수 있습니다.

또한 몇몇 새로운 시각화는 State of JS의 자매 조사인 [State of CSS](https://2019.stateofcss.com/) 에서 빌려왔습니다. 아직 한 번도 보지 못하셨다면 꼭 한 번 확인해보시기를 바랍니다!

### 팀

State of JavaScript 설문은 아래의 팀원들에 의해 만들고 유지됩니다:

- [Sacha Greif](https://twitter.com/sachagreif) (접니다!): 디자인, 글쓰기, 코딩
- [Raphaël Benitte](https://twitter.com/benitteraphael): 데이터 분석, 데이터 시각화

저희의 React/GraphQL 자바스크립트 프레임워크인 [Vulcan.js](http://vulcanjs.org)과 함께, Raphaël의 React 데이터 시각화 라이브러리인 [nivo](https://nivo.rocks)도 꼭 확인해보세요.

### 데이터를 받아보세요

[조사에 사용된 원본 JSON 데이터](https://www.kaggle.com/sachag/state-of-js-2019)도 받아보실 수 있습니다. 여러분만의 방법으로 데이터를 시각화해보셨다면 저희에게 알려주세요!

### 완료율

모든 문항은 응답하지 않고 건너뛸 수 있기 때문에, 다른 것보다 응답 수가 적은 문항도 있었습니다. 그래서 각 문항에는 작은 파이 차트를 넣어 얼마나 많은 응답자가 해당 문항에 답변했는지를 알 수 있게 했습니다.
